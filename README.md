# Cody Hsieh Sand Lab

Single-page falling sand sandbox rebuilt as a pure JavaScript canvas simulation.

## Design

The fresh design doc lives at [docs/sand-game-design.md](docs/sand-game-design.md).

## Stack

- Plain JavaScript
- HTML Canvas 2D
- Webpack for bundling

## Commands

```bash
npm install
npm run dev
npm run build
```

## Notes

- The active runtime is a from-scratch JavaScript 2D simulation.
- Legacy Sandspiel upstream files remain in the repo for reference and licensing context.
